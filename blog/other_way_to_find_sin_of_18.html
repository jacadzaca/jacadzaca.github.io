It turns out that you can find the value of sin(18°) by the following method:

Notice that by the sin addition formula: 

    1° sin(54°) - sin(18°) = 2 * cos((54° + 18°)⁄2) * sin((54° - 18°)⁄2) = 2 * cos(36°) * sin(18°)
Also: 
    sin(54°) - sin(18°) = cos(90° - 54°) - sin(18°) = cos(36°) - sin(18°)
Moreover, by the double cosine formula: 
    2° cos(36°) = cos[(2 * 18)°] = cos2(18°) - sin2(18°)
Now use the Pythagorean identity like this: 
    cos2(18°) - sin2(18°) = 1 - sin2(18°) - sin2(18°) = 1 - 2 * sin2(18°)
Now equate the right-hand side of equation 2° with the right-hand side of statement 1°, and substitute for cos(36°): 
    cos(36°) - sin(18°) = 2 * cos(36°) * sin(18°)
    1 - 2 * sin2(18°) - sin(18°) = 2 * [1 - 2 * sin2(18°)] * sin(18°) = 2 * sin(18°) - 2 * sin3(18°)
    4 * sin3(18°) - 2 * sin2(18°) - 3 * sin(18°) + 1 = 0
    t := sin(18°)
    4t3 - 2t2 - 3t + 1 = 0
Notice that for t = 1 the equation is satisfied, so: 
    4t3 - 2t2 - 3t + 1 = (t - 1)(4t2 + 2t - 1) = 0 <=> t = 1 or 4t2 + 2t - 1 = 0
    4t2 + 2t - 1 = 0
    Δ = 4 + 4 * 4 = 4 * 5
    √Δ = 2√5
    t1 = (-2 + 2√5)⁄8 = (-1 + √5)⁄4
    t2 = (-2 - 2√5)⁄8 = (-1 - √5)⁄4
Now notice that 1 > t > 0, because sin(x) is an increasing function when 90 > x > 0 and sin(90°) = 1 > sin(18°) > sin(0°) = 0, so t = sin(18°) = (-1 + √5)⁄4 , since that's the only solutions that meets the criteria

While this method works, it's not easy to come up with the first equation. I stumbled on it by pure coincidence. I personally prefer the geometrical way of finding the value of sin(18), because it strikes me as the more obvious way of doing it.
